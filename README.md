# ORION Static HTML

This folder is the non-React HTML5 version of the `site-r` landing page.

## Local use

Open `index.html` directly in a browser.

## GitHub Pages

Publish the contents of this folder as a static site.

Options:

1. Put this folder's contents at the repository root for the Pages branch.
2. Or configure GitHub Pages to publish from the `site-r-html` folder if your repo setup allows it.

## Notes

- Images are stored locally in `assets/images`.
- Icons are served from the local `assets/lucide.min.js` bundle.
- Styling uses the Tailwind CDN script, so an internet connection is still needed for CSS generation unless you later replace it with a compiled local stylesheet.
